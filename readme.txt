

Build and run application

Build files
  1. Build pcdNaN.cpp:
    a. run "cp CMakeListsPCDNAN.txt CMakeLists.txt"
    b. run "cd build"
    c. run "cmake ../"
    d. run "make"
  
   2. Build	openni_grabber.cpp:
    a. run "cp CMakeListsOpenniGrabber.txt CMakeLists.txt"
    b. run "cd build"
    c. run "cmake ../"
    d. run "make"
    
   3. Build iterative_closest_point.cpp:
    a. run "cp CMakeListsIcp.txt CMakeLists.txt"
    b. run "cd build"
    c. run "cmake ../"
    d. run "make"
    
Run application
  1. run "cd build"
  2. run  "./openni_grabber"
  3. run "./pcdNaN"
  4. run "./iterative_closest_point.cpp"
